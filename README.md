## Recommender System
### General / Traditional RecSys

> *RecSys paradigm: \
ML(hand-craft features) -> Matrix Factorization (MF) -> Deep Learning Alternatives (RNN, auto-encoder, attention, etc) -> **Graph Convolutional Network (GCN) (Most popular now!)***
